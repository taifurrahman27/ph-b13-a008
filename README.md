# 🚀 SkillSphere

## 📌 Project Name
SkillSphere — Online Learning Platform

---

## 🎯 Purpose
SkillSphere is a modern online learning platform designed to help students upgrade their skills in web development, programming, and design.  
It provides structured courses, instructor guidance, and real-world project-based learning experiences.

The goal of this project is to make learning interactive, accessible, and career-focused.

## ✨ Key Features

- 🎓 Browse and explore modern tech courses
- 🔍 View detailed course information
- 🔒 Protected course details page (login required)
- 👤 User authentication (Login / Signup)
- 🧑 My Profile page with update feature (name & image URL)
- 📚 Trending & Popular courses section
- 🎥 Hero banner with smooth animations
- 🧑‍🏫 Instructor showcase section
- 📱 Fully responsive design (mobile, tablet, desktop)
- ⚡ Modern UI with dark gradient theme
- 🎨 Smooth animations and interactive UI components
